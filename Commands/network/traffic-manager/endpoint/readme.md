# [Group] _network traffic-manager endpoint_

Manage Azure Traffic Manager end points.

## Commands

- [create](/Commands/network/traffic-manager/endpoint/_create.md)
: Create a traffic manager endpoint.

- [delete](/Commands/network/traffic-manager/endpoint/_delete.md)
: Delete a traffic manager endpoint.

- [show](/Commands/network/traffic-manager/endpoint/_show.md)
: Get the details of a traffic manager endpoint.

- [show-geographic-hierarchy](/Commands/network/traffic-manager/endpoint/_show-geographic-hierarchy.md)
: Get the default geographic hierarchy used by the geographic traffic routing method.

- [update](/Commands/network/traffic-manager/endpoint/_update.md)
: Update a traffic manager endpoint.

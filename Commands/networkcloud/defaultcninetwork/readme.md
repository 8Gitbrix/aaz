# [Group] _networkcloud defaultcninetwork_

Manage default CNI network

## Commands

- [create](/Commands/networkcloud/defaultcninetwork/_create.md)
: Create a new default CNI network or update the properties of the existing default CNI network.

- [delete](/Commands/networkcloud/defaultcninetwork/_delete.md)
: Delete the provided default CNI network.

- [list](/Commands/networkcloud/defaultcninetwork/_list.md)
: List default CNI networks in the provided resource group or subscription.

- [show](/Commands/networkcloud/defaultcninetwork/_show.md)
: Get properties of the provided default CNI network.

- [update](/Commands/networkcloud/defaultcninetwork/_update.md)
: Update tags associated with the provided default CNI network.
